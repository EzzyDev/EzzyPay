# Constants

## Statuses

- 2 - Успех
- 3 - Ошибка

```json
"data": {
  "request_id":"c02e79a2-d30c-46e3-b3dd-c0e6a987acf8",
  "status": 2,
  "err_msg": null
}
```

## Errors

- 3000 - Insufficient funds on your account, contact support
- 4000 - An error has occurred, please contact support

```json
"data": {
  "request_id":"c02e79a2-d30c-46e3-b3dd-c0e6a987acf8",
  "status": 3,
  "err_msg": 3000
}
```
